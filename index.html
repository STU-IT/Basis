Velkommen til Basis Programmering

Basis Programmering skal fungere som en introduktion til alt havd vi laver på Programmeringslinien. Derfor er forløbet bygget med en række korte introducerende nedslag på forskellige emner der er typiske for programmering.

De overordnede emner vi skal igennem:

  • Introduktion til C
  • Arbejde med GIT og GitHub
  • Php på webserveren
  • En relationsdatabase
  • JavaScript i browseren

Plan

Den detalierede plan for forløbet tilpasses undervejs, og for hver gang forløbet afholdes. forløbig er 2016 udagven tilgengelig.
